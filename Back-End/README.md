## 🚀 Quick Start

### Prerequisites

| Tool | Version |
|------|---------|
| .NET SDK | 8.0+ |
| SQL Server | 2019+ |
| Visual Studio / Rider / VS Code | Latest |

### 1. Clone & Restore

```bash
git clone <repo-url>
cd Back-End
dotnet restore
```

### 2. Configure Database Connection

Edit `AIStudyHub.API/appsettings.json`:

```json
"ConnectionStrings": {
  "DefaultConnection": "Server=.;Database=AIStudyHub;Trusted_Connection=True;TrustServerCertificate=True"
}
```

### 3. Create Database & Scaffold Entities

> ⚠️ **Database must be created in SQL Server first** (see `DATABASE_DESIGN_GUIDE.md`).

**Package Manager Console** (set Default Project to `AIStudyHub.DAL`):

```powershell
Scaffold-DbContext "Server=.;Database=AIStudyHub;Trusted_Connection=True;TrustServerCertificate=True" Microsoft.EntityFrameworkCore.SqlServer -OutputDir Entities -Context AppDbContext -ContextDir . -DataAnnotations -Force
```

**Or .NET CLI:**

```bash
dotnet ef dbcontext scaffold "Server=.;Database=AIStudyHub;Trusted_Connection=True;TrustServerCertificate=True" Microsoft.EntityFrameworkCore.SqlServer --output-dir Entities --context AppDbContext --context-dir . --data-annotations --force --project AIStudyHub.DAL --startup-project AIStudyHub.API
```

### 4. Post-Scaffolding Steps

1. Update `GenericRepository<T>` constructor: change `DbContext` → `AppDbContext`
2. Update all repository implementations to inject `AppDbContext`
3. Uncomment `AddDbContext<AppDbContext>` in `Program.cs`
4. Uncomment `IGenericRepository<>` registration in `Program.cs`

### 5. Run

```bash
cd AIStudyHub.API
dotnet run
```

Swagger UI: **https://localhost:7xxx** (port shown in console)

## 🛡️ Architecture

Strict **3-Layer Architecture** with separation of concerns:

| Layer | Project | Responsibility |
|-------|---------|----------------|
| **Presentation** | `AIStudyHub.API` | HTTP handling, auth middleware, Swagger |
| **Business Logic** | `AIStudyHub.BLL` | Business rules, validation, orchestration |
| **Data Access** | `AIStudyHub.DAL` | EF Core DbContext, repositories |
| **Cross-cutting** | `AIStudyHub.Shared` | DTOs, enums, constants, exceptions |
| **Testing** | `AIStudyHub.Tests` | Unit tests with xUnit + Moq |
